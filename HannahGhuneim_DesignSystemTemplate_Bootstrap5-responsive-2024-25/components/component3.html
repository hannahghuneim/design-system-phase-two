Button

Description

The button is used to navigate the user to a different part of the website. The use of this button specifically is to send the user to a contact page so they can contact the person who made the website.

Example

Code


                  <div class="container text-center py-3 mx-4">
                  <button 
                      class="btn btn-primary btn-contact start-50 translate-middle-x mb-2"
                      >Contact Me</button>
                  </div>

Accessibility Considerations

The contrast between the white background and the black text makes the text easy for a person to see and it meets the WCAG contrast standards. The text is large enough that the reader can decipher what the button says without straining their eyes. The way that the text is spaced in the middle of the button makes it clear to what the button is meant to show.
